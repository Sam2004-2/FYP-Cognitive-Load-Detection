# Cognitive Load Study Instructions

Thank you for participating.

## Before You Start

- Use a laptop or desktop with a working webcam.
- Use Chrome or Edge (latest version recommended).
- Ensure stable internet.
- Sit in a quiet room with steady lighting.
- Keep your face visible and centered during tasks.

## Consent and Privacy

- You must provide consent on the landing page before continuing.
- The app records study task responses and computed cognitive load features.
- Keep your generated participant ID safe. You must reuse it for Session 2 and delayed testing.

## Session Overview

1. Open the study link.
2. Generate and save your participant ID.
3. Complete camera permission.
4. Complete Session Setup.
5. Run the full study session and NASA-TLX form.
6. The app uploads your session report automatically.

Estimated duration: ~10-15 minutes per session.

## Session 2 Timing

- Session 2 is normally expected at least 24 hours after Session 1.
- If started early, the system logs an override.

## Delayed Test

- Return using the same participant ID.
- Open Delayed Test page and load pending tests from server.
- Complete recognition and cued-recall phases.
- The app uploads delayed results automatically.

## If Upload Fails

If the network upload fails, export fallback JSON from the app and send it to the researcher.
